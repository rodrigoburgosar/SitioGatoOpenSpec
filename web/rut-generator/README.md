# Generador de RUT chileno

Herramienta web basada en React + TypeScript + Tailwind para generar RUTs chilenos válidos en modo individual y masivo.

## Ejecutar en desarrollo

```bash
cd web/rut-generator
npm install
npm run dev
```

## Generar RUTs

- Modo individual: genera un RUT y permite copiarlo.
- Modo masivo: genera una lista y permite descargar un `.xlsx` con columnas `N°`, `RUT`, `RUT_SIN_FORMATO`.
